# Ficha de Treino

## Sobre o Projeto

Essa **Ficha de Treino Full Body** é um sistema desenvolvido para facilitar a organização de exercicios diários, com uma solução prática e visual para gerenciar os exercicios de cada dia.  
Este projeto foi concebido para atender à uma necessidade que eu observei oa longo da minha experiência com academia, onde uma ficha de papel é um tanto inconveniente de ficar carregando, além disso, uma ficha só com os nomes pode ser confuso, de lembrar em qual aparelho fazer ou como executar.

### Problema Resolvido

Pode ser muito difícil manter uma rotina de exercícios consistente devido à falta de organização e clareza sobre quais exercícios realizar em cada dia. 
A ficha de treino tradicional, muitas vezes em papel, pode ser facilmente perdida ou danificada, além de não fornecer informações visuais suficientes para a execução correta dos exercícios.

### Solução Oferecida

Este sistema de Ficha de Treino Full Body oferece uma solução digital e interativa para organizar os exercícios diários. Com fotos, descrições detalhadas e instruções claras para cada exercício. 
Nesse caso, montei baseado na **minha ficha pessoal**.

### **IMPORTANTE:**
Este sistema é apenas uma ferramenta de organização e **não substitui a orientação profissional**. Sempre consulte um profissional de saúde ou um treinador qualificado antes de iniciar qualquer programa de exercícios para garantir que os exercícios sejam adequados para suas necessidades individuais e condições físicas.



## Evolução do Projeto

### Versão Inicial (Old Version)
A primeira versão do sistema era bastante básica, com funcionalidades limitadas, como:

- Somente uma foto de cada exercicio.
- Lista estática de exercícios sem personalização.


### Versão Atual (Atual)
A versão atual do sistema trouxe melhorias significativas, incluindo:

- **Personalização de Treinos:** Um treino completo com fotos, descrições e instruções para cada exercício e cada dia da semana.
- **Dashboard Intuitivo:** Interface simples com botão para exibir as informações e mudar para a outra foto.
- **Integração com API Externa:** Uso de uma API open source para melhorar a experiência do usuário e fornecer funcionalidades adicionais.

## Tecnologias Utilizadas

- **HTML, CSS e JavaScript** para o desenvolvimento front-end.
- **APIs Externas:** 
    - [Exercícios Físicos BD](https://github.com/joao-gugel/exercicios-bd-ptbr/tree/main) para as imagens e descrições dos exercícios.
    - [Free Exercise DB](https://github.com/yuhonas/free-exercise-db) para as imagens.


## Créditos

Este projeto utiliza a [Exercícios Físicos BD](https://github.com/joao-gugel/exercicios-bd-ptbr/tree/main) e [Free Exercise DB](https://github.com/yuhonas/free-exercise-db), duas APIs que foram essenciais para a implementação de cada exercicio, especialmente pelas imagens e JSON bem estruturado.  
Sou muito grato à comunidade por disponibilizar estas ferramentas incríveis.
